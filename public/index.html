This is a simple store API built with Express js

The primary functionality is fetching and filtering data with the query string

Routes

api/products

Basicly you can only make a GET req to this API but you can use filter options

  • featured (accepts true or false)
  • company (accepts company name)
  • name (accepts name)
  • sort (takes in parameter to sort by ...price, name. If '-' included before parameter it will invert the order. You can also add multiple sort by appending with ',')
  • fields
  • numericFilters
  • page (takes in a number representing a page (1 page = 10 products, limit of itmes can be set with ?limit)))
