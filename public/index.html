This is a simple store API built with Express js

The primary functionality is fetching and filtering data with the query string. Database Mongo.db

Product model

name: { type: String, required: [true, "product name must be provided"], },
price: { type: Number, required: [true, "product price must be provided"], },
featured: { type: Boolean, default: false, },
rating: { type: Number, default: 4.5, },
createdAt: { type: Date, default: Date.now(), },
company: { type: String, enum: { values: ["ikea", "liddy", "caressa", "marcos"], message: "{VALUE} is not supported", }, },

Routes

api/products

Basicly you can only make a GET req to this API but you can use filter options

  • featured (accepts true or false)
  • company (accepts company name)
  • name (accepts name)
  • sort (takes in parameter to sort by ...price, name. If '-' included before parameter it will invert the order. You can also add multiple sort by appending with ',')
  • fields (accepts property that you want to retrive based on the model)
  • numericFilters ( Works with price and rating. Example ?numericFilter=price>40)
  • page (takes in a number representing a page (1 page = 10 products, limit of itmes can be set with ?limit))
